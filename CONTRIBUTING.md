# Contributing

Thank you for considering contributing to this project!

## Guidelines

- Search previous suggestions before making a new one to avoid duplicates.
- Make an individual pull request for each suggestion.
- Use the following format: `- [Name](link) - Description.`
- Descriptions must start with an uppercase letter and end with a period.
- New entries should be added to the bottom of the relevant category.
- The pull request should include a link to the resource and explain why it should be included.

## How to contribute

1. Fork the repository.
2. Add your changes to the list.
3. Add yourself to `CONTRIBUTORS.md` (optional).
4. Submit a pull request.
